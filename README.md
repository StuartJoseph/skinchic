#SkinChic

This is an ecommerce beauty store built with HTML, CSS and Javascript.

**This store has the ability to:**

*display a dynamic product page
*add items to dynamic cart
*alert user what their cart total is each time they add an item
*discount codes that update cart total
*deliver options that updates cart total
*order reference number for confirmed orders

##Installation:

*download the skincheck folder
*open the index.html file in your browser
